Hidden System Settings for Flyme

Feb 18, 2017

This app allows you to access system settings that are concealed or realized by manufacturers. Due to system settings are os-version- and vendor-dependent, not all functions (especially Shortcuts ) will work properly on your system with different Android or patch versions.

Table of Contents

  1. Features that don’t require root
  2. Features that require root

Features that don’t require root

The following features are unlocked by this app:

  • Explore components(activities, services, receivers, and providers) of system and third-party apps.
  • Start exported activities (https://developer.android.com/guide/components/activities.html) of other apps
  • Delete/modify saved access points (WiFi), even when they are out of range.
  • Change the priorities of access points
  • Shortcuts for hidden settings of Flyme:
    • Personal dictonary
    • Spell checker
    • Printing Settings
    • Trusted Credentials (disable security certificates)
    • Apps with Usage Access
    • Smart network switch
    • Common data service
    • MTK engineer mode, MTK logger, Location EM2, YGPS

Features that require root

  • Disable/freeze apps (system and 3rd party) and their certain components (activities, services, receivers, and providers),

  • View and share saved WiFi passwords
